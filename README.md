# GestorLenoJS

## Descripción
Proyecto desarrollado en equipo como parte de la materia de **Aplicaciones Web Orientadas a Servicios** en la Universidad Tecnológica de San Juan del Río.  
Se trata de un gestor web construido con **Node.js, Express y EJS**, que implementa rutas, modelos y vistas para manejar datos de manera dinámica.

## Tecnologías utilizadas
- **Node.js**
- **Express**
- **EJS (Embedded JavaScript Templates)**
- **CSS**
- **MySQL** (para la base de datos)

## Estructura del proyecto
- `models/` → Definición de modelos de datos.
- `routes/` → Rutas de la aplicación.
- `middlewares/` → Funciones intermedias para validaciones.
- `views/` → Plantillas EJS para la interfaz.
- `public/` → Archivos estáticos (CSS, imágenes).
- `bd/` → Scripts de base de datos.
- `index.js` → Punto de entrada de la aplicación.
- `package.json` → Dependencias y configuración.

## Instalación y uso

1.- Clonar el repositorio
git clone https://github.com/rakerito/GestorLenoJS.git

2.- Instalar dependencias
npm install

3.- Ejecutar la aplicación
npm start
